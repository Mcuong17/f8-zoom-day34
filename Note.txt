2. SSR và CSR là gì?
- SSR: Server side rendering
       Tạo ra HTML hoàn chỉnh ở phía Server
       -- KeyWord: Trả về HTML, CSS, JS đầy đủ
- CSR: Client side rendering
        Tạo ra HTML hoàn chỉnh ở phía Client
        -- KeyWord: Trả về HTML tối thiểu, thường chỉ gồm
        1. <div id="root"></div>
        2. script src"js-file.js"

Khác nhau ở HTML nhận về
SSR sẽ trả cả ra ở lần get đầu tiên
CSR thì chỉ trả về div root và inner dom vào
